# PDFYandexVisionBlockV1 — Перспективы развития

## Текущее состояние (v1)

На данный момент блок:
- Распознаёт PDF через Yandex Vision OCR (Document Recognition + Text Detection fallback)
- Определяет 8 типов российских документов
- Извлекает реквизиты: номер, дата, сумма, ИНН, КПП, контрагенты, НДС, скидки
- Извлекает табличные данные (товарные позиции)
- Поддерживает 3 схемы аутентификации
- Возвращает структурированный результат (dict/JSON)
- Обрабатывает ошибки с платформенными уведомлениями
- **Протестирован на 350 реальных PDF** с точностью 89-96% по ключевым реквизитам

---

## Векторы развития

### 1. Интеграция Yandex Flash — диалоговый Q&A

**Статус**: приоритетная задача.

Yandex Flash — облегчённая модель Vision API, работающая в 2-5× быстрее базовой. Открывает сценарий **«Спроси документ»**: пользователь задаёт вопросы по PDF и мгновенно получает ответы.

Подробнее — [YANDEX_FLASH.md](YANDEX_FLASH.md).

### 2. Расширение типов документов (8 → 20+)

| Приоритет | Тип документа | Сложность |
|-----------|---------------|-----------|
| Высокий | Счёт-проформа | Низкая |
| Высокий | Товарная накладная ТОРГ-12 | Средняя |
| Высокий | Акт КС-2, КС-3 | Средняя |
| Средний | Приходный/расходный ордер | Средняя |
| Средний | Транспортная накладная ТТН | Высокая |
| Низкий | Доверенность М-2 | Средняя |
| Низкий | Международный инвойс | Средняя |

### 3. Мультистраничные документы

Разделение первой страницы (реквизиты) и последующих (таблицы), сквозная нумерация позиций, итоги с последней страницы.

### 4. Верификация и cross-валидация

- Контрольная сумма ИНН (10 и 12-значных)
- Валидация КПП (формат + код налоговой)
- Сверка суммы товарных позиций с итоговой суммой
- Confidence score из API Yandex Vision
- Подсветка расхождений в `warnings`

### 5. Поддержка изображений и сканов

PNG, JPEG, TIFF — в том числе многостраничные. Предобработка: автоповорот, бинаризация, коррекция перспективы (OpenCV).

### 6. Пакетная обработка

Режим «взять все PDF из папки», параллельные запросы к Vision API, прогресс-бар.

### 7. Экспорт в целевые форматы

| Формат | Применение |
|--------|------------|
| `1C_JSON` | Загрузка в 1С:Предприятие |
| `CSV` | Табличный экспорт |
| `XLSX` | Excel с форматированием |

### 8. Self-hosted режим (Privacy)

Локальный OCR (Tesseract) для случаев, когда документы нельзя отправлять в облако.

### 9. Дообучение на пользовательских шаблонах

Разметка 5-10 примеров → запоминание bounding box'ов → сверхточное извлечение для повторяющихся форм.

### 10. Мульти-провайдер OCR

Абстрактный слой `OCRProvider`: Yandex Vision, Tesseract, Google Vision, Azure Form Recognizer.
